# 💊 Extraindo Insights de Feedbacks de Clientes do Setor Farmacêutico com IA

Projeto desenvolvido durante o curso da DIO utilizando conceitos de Engenharia de Prompt para construir um modelo de análise de feedbacks de clientes com apoio de Inteligência Artificial.

## 📖 Sobre o Projeto
Este projeto foi desenvolvido como parte de um desafio prático da DIO com o objetivo de aplicar conceitos de Engenharia de Prompt na construção de instruções claras e estruturadas para Inteligência Artificial.

Como estudo de caso, foi escolhido o setor farmacêutico em vez do exemplo bancário proposto no desafio. O projeto demonstra como uma IA pode ser orientada a analisar feedbacks de clientes para identificar padrões, oportunidades de melhoria e gerar insights que apoiem a tomada de decisão.

## 🎯 Objetivo
Desenvolver um prompt completo e bem estruturado capaz de orientar uma Inteligência Artificial na análise de feedbacks de clientes de uma rede farmacêutica.

O objetivo é transformar comentários dos consumidores em informações úteis para apoiar decisões estratégicas relacionadas à qualidade do atendimento, experiência do cliente, logística, disponibilidade de produtos e melhoria contínua dos serviços.🚀

## 🚀 Como Utilizar
Leia o prompt disponível na pasta docs/prompt-final.md.
Utilize uma ferramenta de Inteligência Artificial, como o ChatGPT ou outro modelo de linguagem compatível.
Cole o prompt juntamente com uma base de feedbacks de clientes.
Solicite que a IA realize a análise conforme as instruções definidas.
Analise os resultados obtidos, incluindo os insights, classificações, padrões identificados e recomendações geradas.

O prompt pode ser adaptado para diferentes empresas, produtos ou segmentos do mercado apenas alterando o contexto e os dados fornecidos.

## 📌 Resultado Esperado
Ao executar o prompt, espera-se que a Inteligência Artificial seja capaz de:

Identificar os principais temas presentes nos feedbacks.
Classificar os comentários por sentimento (positivo, negativo ou neutro).
Detectar reclamações recorrentes e oportunidades de melhoria.
Apontar evidências presentes nos comentários analisados.
Organizar os resultados em um resumo executivo e em tabelas de fácil interpretação.
Sugerir ações práticas para auxiliar gestores na tomada de decisão.

O resultado final deve ser claro, organizado, objetivo e baseado exclusivamente nas informações fornecidas, respeitando boas práticas de análise de dados e proteção de informações sensíveis.
